Adventures in Computer Science

A year ago I took Udacity's online web development course. This course was about using Python and Google App Engine to create web applications. This website is based on the things I learnt in the course.

It has a Caesar Cypher page, an Ascii art page and a Blog. Both the Ascii art and the blog make use of a database and basic sql to retrieve and sort the data from the databases.

The Ascii art page also has a map generated using the Google maps api. It shows the locations of people when they submit some art.

Crafting Logic

After creating my Adventures website I felt I needed a project that was more individual to me and one that could expand as I learn more. I have always enjoyed a variety of crafts, particularly those that are underpinned by patterns and logic.

As well as a blog like my Adventures site it also has a gallery and moderated comments.
